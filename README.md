## 🍺 Brasserie du Vénasque

Design system complet pour la **Brasserie du Vénasque** — brasserie artisanale au cœur des Pyrénées (Pays de Luchon, Comminges, Haute-Garonne).

📄 [Consulter le design system →](./brasserie-du-venasque/README.md)

**Contenu :**
- Identité de marque & valeurs visuelles
- Palette de couleurs (primaires + 13 couleurs par bière + 4 saisonnières)
- Typographies (Anton, Montserrat, Inter)
- Iconographie & illustrations linéaires
- Composants signature
- Style photographique
- Ton de voix par cible
- Layouts pour newsletters Brevo (B2C/B2B), fiches produits, PLV, site web
- Variables CSS prêtes à l'emploi
